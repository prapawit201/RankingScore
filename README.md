# Ranking Score leaderboard

##  Requirement

 - NodeJS 10 latest. [Doc](https://nodejs.org/en/docs/)
 - Angular7  [Doc]([https://angular.io/docs](https://angular.io/docs))
 - MongoDB.  [Doc](https://docs.mongodb.com/manual/reference/program/mongod/)
